This project is a simple Education Management System built using Core Java and enhanced with a Graphical User Interface (GUI). It helps manage student records, teacher details, and lecturer data, making administrative tasks easier and more efficient for educational institutions.

Tools Used
Language: Java (Core)
IDE: BlueJ
GUI Library: javax.swing

Project Description
Originally developed as a console-based system, this project was later upgraded with a GUI to improve usability. The system follows proper Object-Oriented Programming (OOP) principles, making it modular and easy to maintain.

Key functionalities include:
Adding and managing student, teacher, and lecturer information\
User-friendly graphical interface
Organized class design using OOP principles
